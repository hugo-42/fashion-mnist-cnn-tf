Fashion‑MNIST — CNN (TensorFlow)

Projet de classification d'images (10 classes, 28×28). Notebook exécutable dans Google Colab, code open‑source sur GitHub.

Ouvrir dans Colab Voir le dépôt GitHub Voir le notebook

Objectif

Construire un petit réseau de neurones convolutionnel (CNN) pour reconnaître des vêtements du dataset Fashion‑MNIST.

Accuracy val : 91%
Accuracy test : 90%

Stack

  • TensorFlow / Keras
  • NumPy, Matplotlib
  • scikit‑learn
  • Jupyter Notebook

Données

  • Source : tensorflow.keras.datasets.fashion_mnist
  • Images 28×28 en niveaux de gris, 10 classes
  • Ensembles d'entraînement et de test fournis

Modèle

  • Convolutions → ReLU → MaxPool (×2)
  • Dropout
  • Denses (FC) → Softmax (10 classes)
  • Optimiseur Adam, perte : sparse_categorical_crossentropy
